Desarrollo Web Frontend Profesional

Este libro ha sido creado con la experiencia de haber realizado varias plantillas de extensa complejidad para el sitio web wrapbootstrap donde el autor ha vendido más de 6000 plantillas.

Características

  • Aprende a utilizar las mejores herramientas para desarrollo web.
  • Conoce los método para estructurar mejor tu proyecto.
  • Agiliza tu trabajando automatizando tareas con Gulp.
  • Aprende a usar NodeJS, Bower y Gulp para el desarrollo frontend.
